## Как использовать

### 1. Добавить в меню настроек

Откройте файл настроек приложения и добавьте пункт меню:

```swift
// В SettingsController.swift или аналогичном файле
let ghostModeItem = ItemListDisclosureItem(
    presentationData: presentationData,
    title: "👻 Ghost Mode",
    label: "",
    sectionId: self.section,
    style: .blocks,
    action: {
        let controller = ghostSettingsController(context: context)
        arguments.pushController(controller)
    }
)
```

### 2. Интеграция функций

Каждая функция требует интеграции в соответствующие части кода:

#### Скрытие онлайн-статуса
```swift
// В файле обработки статуса
if getGhostSettings(transaction: transaction).hideOnlineStatus {
    // Не отправлять статус онлайн
    return
}
```

#### Сохранение удалённых сообщений
```swift
// При получении события удаления
if getGhostSettings(transaction: transaction).saveDeletedMessages {
    // Сохранить сообщение в отдельную таблицу
    saveDeletedMessage(message, transaction: transaction)
}
```

#### Отключение прочтения
```swift
// При отправке read receipts
if getGhostSettings(transaction: transaction).disableReadReceipts {
    // Не отправлять подтверждение прочтения
    return
}
```

## Примечания

- Все настройки сохраняются локально в Postbox
- Функции работают только на клиентской стороне
- Некоторые функции могут требовать модификации сетевых запросов
- Premium функции работают локально без серверной подписки

## Безопасность
